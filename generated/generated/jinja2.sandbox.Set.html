jinja2.sandbox.Set¶

class jinja2.sandbox.Set(iterable=None)[source]¶

Mutable set class.

Methods¶

__init__([iterable]) Construct a set from an optional iterable.
add(element) Add an element to a set.
clear() Remove all elements from this set.
copy() Return a shallow copy of a set.
difference(other) Return the difference of two sets as a new Set.
difference_update(other) Remove all elements of another set from this set.
discard(element) Remove an element from a set if it is a member.
intersection(other) Return the intersection of two sets as a new set.
intersection_update(other) Update a set with the intersection of itself and another.
issubset(other) Report whether another set contains this set.
issuperset(other) Report whether this set contains another set.
pop() Remove and return an arbitrary set element.
remove(element) Remove an element from a set; it must be a member.
symmetric_difference(other) Return the symmetric difference of two sets as a new set.
symmetric_difference_update(other) Update a set with the symmetric difference of itself and another.
union(other) Return the union of two sets as a new set.
union_update(other) Update a set with the union of itself and another.
update(iterable) Add all values from an iterable (such as a list or file).
